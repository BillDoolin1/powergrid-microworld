# Power Grid Microworld

An interactive energy planning simulation where students build Ireland's
power grid from 2026 to 2050. Designed as a classroom tool for live workshops.

**[Play here](https://billdoolin1.github.io./powergrid-microworld/)**

---

## About

Players invest in a mix of energy sources while balancing a budget, hitting
capacity targets, and cutting greenhouse gas emissions across three levels of
increasing difficulty. Built for a younger audience with gamification baked
in to keep things engaging.

---

## Features

- 3 levels progressing toward 2050
- Budget and GHG emissions management
- Timer-based decision making
- 6 unlockable achievements
- Best score saved locally via localStorage
- No login or install needed

---

## Built With

- HTML5, CSS3, JavaScript (ES6+)
- Hosted on GitHub Pages
- Local development via Live Server

---

## Workshop Use

Share the link with students and go. Teams compete to meet Ireland's climate
targets, then present and justify their decisions at the end of the session.
